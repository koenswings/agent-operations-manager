## Key Design Decision

Each Tabby profile uses `tmux new-session -A -s <session> -c <dir> 'claude; exec bash -l'`. The `-A` flag makes each tab self-sufficient: attach if session exists, create if not. No dependency between tabs — opening any single tab works correctly regardless of whether the others are running.
