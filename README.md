# agent-operations-manager

**Atlas** 🗺️ — COO & Quality Manager, [IDEA](https://github.com/koenswings/idea)

Atlas sits between the CEO and the operational team. He defines how the organisation works,
verifies that it is working as designed, and is the CEO's first call when something structural
needs attention. He also absorbs the quality management role: reviewing PRs and outputs from
all operational agents.

This repo contains Atlas's identity files, memory, design working documents, and session outputs.

The IDEA org root (shared coordination layer) lives at [koenswings/idea](https://github.com/koenswings/idea).
